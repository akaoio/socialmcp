# install.sh + scripts — zero-manual-setup onboarding

## Thay đổi

### Vấn đề

Dev mới `npm install` xong vẫn thiếu:
- Playwright Chromium (`playwright install chromium` chưa chạy)
- System packages cho noVNC stack (`xvfb`, `x11vnc`, `websockify`, `novnc`, `chromium`)
- `python3-pycryptodome` (dùng để giải mã cookies Chromium)

Phải làm tay từng bước — không có script hướng dẫn.

### Giải pháp

**`install.sh`** — một lệnh setup mọi thứ:

```bash
./install.sh           # base: npm install + playwright install chromium (mọi dev Linux)
./install.sh --server  # base + noVNC stack + system chromium + pycryptodome
```

**`scripts/startnovnc.sh`** — start/stop toàn bộ noVNC stack:

```bash
scripts/startnovnc.sh          # khởi động: Xvfb → openbox → x11vnc → websockify
scripts/startnovnc.sh --stop   # dừng tất cả
```

Tự detect thành phần nào chưa chạy trước khi start. Logs tại `/tmp/socialmcp-novnc/`.

**`scripts/extractcookies.js`** — giải mã Facebook cookies từ Chromium profile:

```bash
node scripts/extractcookies.js > /tmp/fb_cookies.json
FACEBOOK_COOKIES=$(node scripts/extractcookies.js) npm test
```

Trước đây phải chạy script Python thủ công, copy ra file, rồi set env var.

### `.gitignore` fix

`scripts/` bị ignore blanket → 3 file mới không commit được.
Xóa dòng `scripts/` khỏi `.gitignore`; giữ các rule cụ thể (`scripts/.chrome-profile/`, `scripts/*.png`, v.v.) vì chúng vẫn cần.

## Tradeoffs

- `install.sh --server` dùng `apt-get` → chỉ chạy được trên Debian/Ubuntu. Nếu sau này cần support macOS sẽ phải thêm `brew` path.
- VNC password set tại `~/.vncpasswd` (ngoài repo) — phải set lại nếu đổi máy.
- `extractcookies.js` hardcode path `~/.config/chromium/Default/Cookies` — đúng cho Linux Chromium, sai nếu dùng Chrome hoặc macOS.
