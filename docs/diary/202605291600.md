# 2026-05-29 — Xóa toàn bộ external runtime dependencies

## Bối cảnh

Session hôm nay bắt đầu từ việc server không start được (`npm start` lỗi) do chưa cài `node_modules`. Từ đó dẫn đến một loạt quyết định refactor lớn.

## Những thay đổi chính

### 1. Chuyển build system từ esbuild sang rollup + terser

**Lý do:** esbuild là devDep nhưng project ưu tiên thuần ESM, terser là minifier chuẩn hơn cho production. Rollup phù hợp hơn với ESM-first codebase.

**Kết quả:**
- `build.js` viết lại hoàn toàn dùng `rollup`, `@rollup/plugin-node-resolve`, `@rollup/plugin-json`, `@rollup/plugin-terser`
- `package.json` root: bỏ `esbuild`, thêm các rollup plugins
- Server bundle: external chỉ `node:*` built-ins, còn lại bundle hết vào

### 2. Loại bỏ toàn bộ external runtime dependencies của server

**Lý do:** Owner không muốn dùng lib ngoài cho runtime. Mục tiêu zero-dep ngoại trừ `@akaoio/zen` (do chính team phát triển).

**Những gì bị xóa:**
- `@modelcontextprotocol/sdk` — thay bằng `src/server/mcp.js` tự viết
- `zod` — schema builder viết lại trong `mcp.js` (API tương thích)
- `ws` npm package — thay bằng `@akaoio/zen/lib/websocket.js`
- `src/server/package.json` và `package-lock.json` — xóa, merge vào root `package.json`

**Những file mới:**
- `src/server/mcp.js` — MCP JSON-RPC server (stdio transport) + schema builder `z`
  - Implements: `initialize`, `ping`, `tools/list`, `tools/call`
  - Schema: `z.string()`, `z.number()`, `z.enum()`, `z.array()`, `.describe()`, `.optional()`, `.int()`, `.min()`, `.max()`
- `src/server/ws.js` (tạm thời) — sau đó xóa khi chuyển sang zen

### 3. Dùng @akaoio/zen cho WebSocket

**Lý do:** zen là lib internal của team, có `lib/websocket.js` implement WS protocol từ TCP thuần, không phụ thuộc `ws` package.

**Thay đổi `bridge.js`:**
```js
// Trước
import { WebSocketServer } from 'ws';
this.wss = new WebSocketServer({ port, host: '127.0.0.1' });

// Sau
import WS from '@akaoio/zen/lib/websocket.js';
const { Server: WebSocketServer } = WS;
const server = createServer();
this.wss = new WebSocketServer({ server });
server.listen(this.port, '127.0.0.1');
```

**Lưu ý:** zen's `WebSocketServer` nhận `{ server }` (HTTP server), không nhận `{ port, host }` trực tiếp.

## Tradeoffs

- `src/server/mcp.js` chỉ implement phần MCP tối thiểu cần thiết. Nếu sau này cần features như `resources`, `prompts`, `sampling` thì cần mở rộng thêm.
- Schema builder không có tất cả validators của zod (chỉ có những gì đang dùng). Thêm khi cần.
- `@akaoio/zen` là runtime dependency nên được bundle vào build output.

## Trạng thái hiện tại

- `node src/server/index.js` — chạy được, không cần `npm install` trong `src/server/`
- `node build.js` — build sạch, không warning
- Root `package.json` là nguồn duy nhất cho tất cả dependencies
